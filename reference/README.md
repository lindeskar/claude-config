# reference/ — war-story archive

Verbatim snapshots of the full rule files as they were before the 2026-07-07
context diet. The lean `rules/*.md` (auto-loaded into every session) keep every
imperative rule; these archives keep the full "(Learned …)" case studies and
long-form procedures that justified them.

This directory is **not** auto-loaded. Read a file here when a rule in
`rules/` needs its evidence or full worked example. Kognic-specific procedures
were moved to the work wiki (`~/Code/_private/work/wiki/`) instead — the lean
rules name the wiki entry where that applies.

New learnings: add the one-line rule to `rules/`, and put the story either in
the matching file here or (if Kognic-specific) in the work wiki.
