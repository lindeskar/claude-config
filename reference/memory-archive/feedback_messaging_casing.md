Use proper capitalization in Slack message drafts — uppercase letters at the start of every sentence.

**Why:** User corrected a draft that used lowercase sentence starts. Even though Slack is casual, proper casing is expected.

**How to apply:** When drafting any Slack message, ensure every sentence starts with an uppercase letter. Don't confuse "casual tone" with "no capitalization."
